Four fully-styled landing-page mockups exploring visual identities that would set ZeroGEX apart from the standard SaaS-dashboard-in-navy-blue crowd. Each is a standalone page — open one, live in it for a minute, see how it feels.

1. The Almanac

Editorial. Serif-heavy. Cream paper, oxblood ink, gold-leaf accents. A trading almanac you'd find in an old broker's library — dropped into 2026 with live data.

Warm cream, chunky bevelled buttons, hand-drawn line illustrations, rotated sticker badges. PostHog-adjacent, but built for gamma traders — with a gecko mascot pulling shift on the trading floor.
