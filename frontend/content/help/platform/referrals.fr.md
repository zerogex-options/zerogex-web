# Parrainage

*Comment fonctionne le programme de parrainage — votre code, votre lien, ce qui compte comme parrainage et comment les mois gagnés s'appliquent à votre facture.*

---

## En quoi consiste le programme de parrainage

Lorsque le programme de parrainage est activé pour votre compte, vous obtenez :

- Un **code de parrainage** — une courte chaîne alphanumérique qui vous est propre.
- Un **lien de parrainage** — la version facile à partager de votre code.

Toute personne qui s'inscrit avec votre code (ou lien) et passe à un forfait payant compte comme un parrainage réussi. Chaque parrainage réussi vous rapporte un crédit sur votre prochaine facture.

## Où trouver votre code

Ouvrez [Account](/account). Le panneau Parrainage affiche votre code, votre lien et votre situation actuelle :

- **Inscriptions totales** — combien de comptes ont utilisé votre code.
- **Conversions totales** — combien d'entre eux se sont abonnés à Basic ou Pro.
- **Mois gagnés** — solde total de crédit accumulé grâce aux parrainages.
- **Mois crédités** — crédit actuellement appliqué sur votre prochaine facture.
- **Crédit sur la prochaine facture** — valeur en dollars du crédit à votre tarif actuel.

## Ce qui compte comme un parrainage réussi

- Le nouvel utilisateur doit s'inscrire avec votre code appliqué.
- Il doit passer à un forfait payant (essai → payant) et terminer au moins un cycle de facturation complet.
- Les auto-parrainages ne sont pas autorisés (détectés automatiquement par le système).
- Le parrain doit avoir une adresse e-mail vérifiée et un forfait payant actif.

## Comment le crédit est appliqué

Les crédits sont appliqués sur votre **prochaine facture Stripe** sous forme de remise. Ils se cumulent — plusieurs parrainages au cours du même cycle s'additionnent.

Les crédits ne sont ni transférables ni remboursables en espèces. Ils expirent si votre compte est fermé.

## Partager votre code

Trois pratiques que nous observons :

- **Lien direct** — le plus simple. Copiez-le depuis la page Account et collez-le.
- **Code dans une publication** — partagez le code lui-même ; les nouveaux utilisateurs le saisissent lors de l'inscription.
- **Message privé au bon moment** — quand quelqu'un dans une communauté de trading pose une question sur le gamma exposure, le lien de parrainage est aussi une réponse.

## Ce que le nouvel utilisateur y gagne

Le nouvel utilisateur bénéficie de la même période d'essai standard (aucune extension via votre code), et toute promotion active sur l'ensemble du site s'applique en plus. Le système de parrainage vous crédite **vous** pour la conversion, pas lui — le programme est conçu comme un remerciement, pas comme un transfert de remise.

## Confidentialité

Votre code vous appartient. La page Account **n'affiche pas** qui l'a utilisé — seulement les totaux. L'identité des nouveaux utilisateurs n'est jamais révélée aux parrains.

## Éligibilité et statut du programme

Le programme est activé par compte selon la configuration. Si vous ne voyez pas de panneau Parrainage sur votre page Account, le programme n'est pas actuellement activé pour votre compte.
