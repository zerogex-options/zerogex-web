ZeroGEX Design System · v1

Color and typography spec

Four themes × two modes. Every row sits on its own --bg-main, so each swatch renders against the ground it actually appears against in-product. Semi-transparent tokens (border alphas, text alphas) show their true composite over that ground.

Color tokens · 8 combos × 30 tokens

Typography · 4 themes × 3 roles (mode-agnostic)

Reading this sheet

  • Each row's cell backgrounds are the theme's --bg-main. A #RRRRRR label alone is a solid value; #RRRRRR · N% is the base color at N% alpha, composited over the row ground.
  • Column groups follow the globals.css token names verbatim: --color-bg-*, --text-*, --border-*, --color-accent*, --color-brand-*, --color-bull / --color-bear, --heat-*, --chart-1..5.
  • Fonts are theme-level (not mode-level); font stacks fall through to system faces where the display face isn't installed locally.
